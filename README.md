Project Description
This project implements a **Content-Based Movie Recommendation System** using machine learning techniques. Movies are recommended based on genre similarity using cosine similarity.

Objectives
- Analyze movie metadata
- Build a content-based recommendation model
- Recommend similar movies
- Develop a web interface using Streamlit

Technologies Used
Python, Pandas, NumPy, Scikit-learn, Streamlit

Dataset
Custom movie dataset inspired by MovieLens

 Recommendation Technique
Content-Based Filtering, Cosine Similarity

How to Run
pip install -r requirements.txt  
streamlit run app.py

Future Scope
- Collaborative filtering
- Hybrid recommendation system
- RMSE and MAE evaluation
